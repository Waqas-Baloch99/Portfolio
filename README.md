## ⚙️ Installation  
```bash
git clone https://github.com/Waqas-Baloch99/Portfolio.git
cd Portfolio
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
